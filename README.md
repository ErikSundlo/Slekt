# Historisk arkiv – Sundlo og Ringset-slektene

Dette repositoriet inneholder et digitalt familiearkiv med slektshistorie, biografier og historiske notater knyttet til Sundlo- og Ringset-slektene.

Nettsiden publiseres med GitHub Pages.

## Innhold

Arkivet inneholder blant annet:

- slektshistorie for Sundlo- og Ringset-slektene
- biografiske tekster om personer i slekten
- historiske notater og erindringer
- materiale nedskrevet og samlet av Harald Sundlo i Asker 1993/94


## Historisk merknad

Arkivet bygger også på notater og erindringer knyttet til Konrad Sundlo, som gir et særpreget innblikk i personer, miljøer og hendelser.

Han var en sammensatt historisk skikkelse, og materialet er derfor tatt inn som kildestoff og bør leses i lys av sin tid.
