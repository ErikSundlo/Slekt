# Historisk arkiv – Sundlo og Ringset-slektene

A digital archive of family history, biographies, and historical documents related to the Sundlo and Ringset families from Norway.

**Live site:** https://eriksundlo.github.io/Slekt/

## About

This archive contains 40+ documents originally written by Harald Sundlo in Asker (1993/94), now digitized and published as a static website via GitHub Pages. Content is primarily in Norwegian, with selected pages translated to English and German.

The documents cover:

- **Sundlo-slekten** – Sundlo family genealogy
- **Ringset-slekten** – Ringset family genealogy
- **Konrad Sundlo forteller** – Konrad Sundlo's personal accounts
- **Politikompaniet** – The Police Company
- **Regiment Nordland** – Regiment Nordland
- **Legioner og bataljoner** – Legions and battalions
- **Formål og bakgrunn** – Purpose and background

## Build

The site is generated by a two-stage Python pipeline:

```bash
pip install beautifulsoup4

# Stage 1: Clean and reformat LibreOffice-exported HTML
python reformat.py

# Stage 2: Enhance with headings, anchors, and table of contents
python enhance.py
```

## Deployment

Static HTML served by GitHub Pages. Push to `master` to deploy.
